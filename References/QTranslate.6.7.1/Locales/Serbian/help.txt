Системске пречице (подразумевано):
Double Ctrl => Приказује главни прозор (са преводом, ако је изабран текст)
Ctrl+Q => Преводи изабрани текст и приказује га у искачућем прозору
Ctrl+Shift+Q => Проналази изабрани текст на интернету
Ctrl+E => Послушај изабрани текст

Главни прозор тастерских пречица:
Ctrl+Enter => Преведи текст
Ctrl+N => Избриши тренутни  превод
Ctrl+H => Прикажи историјат превода
Ctrl+K => Прикажи виртуелну тастатуру
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => Пребаци се на следећи/претходни/1..9th сервис за превођење
Shift+Esc => У почетно пар језика за аутоматско откривање
Ctrl+I => Обрни смер превођења
Ctrl+Space => Прикажи избор са списком могућности за унос речи (аутоматско довршавање)
F1 => Прикажи Помоћ
Програм има два режима превода текста избором са мишем (како да га укључите, види доле):
  1) Прикажи икону (изаберите текст у било ком програму -> икона програм се појављује поред показивача; кликните на икону -> искачући прозор се појављује са преводом изабраног текста);
  2) Прикажи превод (изаберите текст у било ком програму -> искачући прозор се појављује са преводом изабраног текста).

Дејство:
  - Кликните левим тастером миша на икону програма у системској палети да бисте укључили / искључили режим миша за избор (види горе);
  - Двапут кликните левим тастером миша на икону програма у системској палети за приказ главног прозора;
  - Двапут кликните левим тастером миша на заглавље у искачућем прозору да бисте отворили главни прозор са тренутним преводом;
  - Кликните левим тастером миша на име сервиса у искачућем прозору да промените тренутни сервис и поново преведете текст.
