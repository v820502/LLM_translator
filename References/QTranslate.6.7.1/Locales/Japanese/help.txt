システム全体ホットキー (デフォルト値):
Double Ctrl => (可能ならば選択した文章で)メインウィンドウを表示 
Ctrl+Q => 選択した文章を翻訳しポップアップウィンドウ内に表示する
Ctrl+Shift+Q => 選択した文章をWEB検索
Ctrl+E => 選択した文章を音声再生する

ホットキー:
Ctrl+Enter => メインウィンドウ内に翻訳文章を表示
Ctrl+N => 現在の翻訳をクリア
Ctrl+H => 翻訳履歴ダイアログを表示する
Ctrl+K => 仮想キーボードを表示する
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => 次の/前の/1～9番目の 翻訳サービスの切り替え
Shift+Esc => 言語の組み合わせを自動選択(Auto-Detect)にリセット
Ctrl+I => 翻訳方向を逆転
Ctrl+Space => 提案/自動補完ドロップダウンメニューを表示する
F1 => ヘルプ

このプログラムには、マウス選択による2つの翻訳モードがあります (切り替え方法については以下を参照してください):
  1) アイコン表示 (任意のプログラム上でテキストを選択すると、カーソルの近くにアイコンが表示されます。アイコンをクリックするとポップアップウィンドウに選択したテキストの翻訳が表示されます);
  2) 翻訳の表示 (任意のプログラム上でテキストを選択すると、即座にポップアップウィンドウに選択したテキストの翻訳が表示されます).
    
Actions:
  - システムトレイのプログラムアイコンの左クリックで、マウス選択モードをオン/オフにします (上記参照);
  - システムトレイのプログラムアイコンの左ダブルクリックで、メインウィンドウを表示します。;
  - ポップアップウィンドウのヘッダーを左ダブルクリックで、現在の翻訳テキストの内容でメインウィンドウを開きます。;
  - ポップアップウィンドウ内のサービス名ボタンをクリックすると、サービスを変更しテキストを再翻訳します。
