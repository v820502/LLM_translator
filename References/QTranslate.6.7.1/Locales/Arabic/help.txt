نظام اختصارات المفاتيح الساخنة(القيم الافتراضية(:
Double Ctrl => إظهار الإطار الرئيسي (مع ترجمة النص المحدد، أن وجد(
Ctrl + Q => ترجمة النص المحدد وإظهاره في نافذة منبثقة
Ctrl+Shift+Q => إظهر نافذة القاموس
Ctrl + E => الاستماع إلى النص المحدد

اختصارات لوحة المفاتيح في الإطار الرئيسي:
Ctrl + Enter => ترجمة النص
Ctrl + N => تنظيف الترجمة الحالية
Ctrl + H => إظهار مربع الحوار لتأريخ الترجمة
Ctrl + K => إظهار لوحة المفاتيح الافتراضية
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => التبديل إلى خدمة الترجمة التالي/السابق /1..9th
shift + Esc => إعادة الكشف التلقائي عن اللغة
Ctrl + I => عكس اتجاه الترجمة
Ctrl + Space  => إظهار القائمة المنسدلة "الإكمال التلقائي"/اقتراح
F1 => إظهار التعليمات

هناك وضعان للترجمة بالماوس الاختيار (كيفية تشغيله، راجع أدناه):
  1) إظهار الايقونة (تحديد نص في بعض البرامج، ورمز البرنامج سوف يظهر بالقرب من المؤشر. انقر على الأيقونة، وسوف تظهر نافذة منبثقة مع ترجمة النص المحدد)؛
  2) إظهار الترجمة (تحديد نص في بعض البرامج، ونافذة منبثقة مع ترجمة النص المحدد سوف تظهر فورا)‎.

الإجراءات:
  -انقر فوق زر الماوس الأيمن على رمز البرنامج في علبة النظام لـ تشغيل/إيقاف وضع التحديد الماوس (انظر أعلاه)؛
  --انقر نقراً مزدوجاً فوق زر الماوس الأيمن على رمز البرنامج في علبة النظام لإظهار الإطار الرئيسي؛
  --انقر نقراً مزدوجاً فوق زر الماوس الأيسر على رأس الإطار المنبثق فتح الإطار الرئيسي مع الترجمة الحالية؛
  -انقر فوق زر الماوس الأيسر على اسم الخدمة في نافذة منبثقة لتغيير الخدمة الحالية وإعادة--ترجمة النص.
