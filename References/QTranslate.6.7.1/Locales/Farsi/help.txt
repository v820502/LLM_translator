کلید میانبر های سراسری (مقادیر پیش فرض):
دو مرتبه کلیک => نمایش پنجره اصلی (با ترجمه متن انتخاب شده،اگر وجود داشته باشد)
Ctrl+Q => ترجمه متن انتخاب شده و نمایش آن در پنجره پاپ آپ
Ctrl+Shift+Q => نمایش پنجره واژه نامه
Ctrl+E => گوش دادن به متن انتخاب شده

کلید میانبر های پنجره اصلی:
Ctrl+Enter => ترجمه متن
Ctrl+N => پاکسازی ترجمه فعلی
Ctrl+D => نمایش واژه نامه
Ctrl+Alt+1..9 => نمایش واژه نامه با سرویس ترجمه 1 تا 9
Ctrl+H => نمایش تاریخچه ترجمه
Ctrl+K => نمایش صفحه کلید مجازی
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => رفتن به سرویس ترجمه 1 تا 9 بعدی و قبلی
Shift+Esc => ریست جفت زبان به تشخیص خودکار
Ctrl+Shift+1..9 => جفت زبان را انتخاب می کند
Ctrl+I => تعویض جهت ترجمه
Ctrl+Space => نمایش منوی پیشنهادات/کامل کردن خودکار
F1 => نمایش راهنما
F11 => خاموش/روشن کردن حالت تمام صفحه
Alt+Left Arrow => رفتن به ترجمه قبلی
Alt+Right Arrow => رفتن به ترجمه بعدی
Ctrl+Up => کپی ترجمه به کادر ورودی متن

سه روش ترجمه با انتخاب ماوس وجود دارد:
  1) نمایش آیکون (انتخاب متن در یک برنامه -> آیکون برنامه نزدیک مکان نما ظاهر خواهد شد. روی آیکون کلیک کرده -> پنجره پاپ آپ با ترجمه متن انتخاب شده ظاهر خواهد شد);
  2) نمایش ترجمه (انتخاب متن در یک برنامه -> پنجره پاپ آپ با ترجمه متن انتخاب شده فوراً ظاهر خواهد شد);
  3) نمایش ترجمه و خواندن (همانند قبلی بعلاوه خواندن با صدای بلند متن انتخاب شده). 

اقدامات:
  - برای روشن/خاموش کردن حالت انتخاب ماوس روی آیکون برنامه در کنار ساعت کلیک چپ کنید (بالا را مشاهده کنید);
  - دو مرتبه کلیک روی آیکون برنامه در کنار ساعت برای باز کردن پنجره اصلی;
  - روی سربرگ پنجره پاپ آپ دو مرتبه کلیک کرده تا پنجره اصلی با ترجمه فعلی باز شود;
  - کلیک چپ روی نام سرویس در پنجره پاپ آپ،سرویس فعلی را تغییر داده و متن را مجدد ترجمه می کند.
