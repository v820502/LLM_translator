پۈتۈن تېز كۇنۇپكىلار ‎(سۈكۈتتىكى قىممەتلىرى):
قوش Ctrl => (تاللىغان تېكىست تەرجىمىسى بىلەن) ئاساسىي كۆزنەكنى كۆرسىتىش
Ctrl+Q => تاللىغان تېكىستنى تەرجىمە قىلىش ۋە سەكرىمە كۆزنەكتە كۆرسىتىش
Ctrl+Shift+Q => لۇغەت كۆزنىكىنى كۆرسىتىش
Ctrl+E => تاللىغان تېكىستنى ئاڭلاش

ئاساسىي كۆزنەك تېز كۇنۇپكىلىرى:
Ctrl+Enter => تېكىستنى تەرجىمە قىلىش
Ctrl+N => نۆۋەتتىكى تەرجىمىنى تازىلاش
Ctrl+D => لۇغەتنى كۆرسىتىش
Ctrl+Alt+1..9 => 1-9غىچە تاللانغان تەرجىمە مۇلازىمىتىدىن لۇغەتتە كۆرسىتىش
Ctrl+H => تەرجىمە تارىخىي ئۇچۇرىنى كۆرسىتىش
Ctrl+K => مەۋھۇم كۇنۇپكا تاختىسىنى كۆرسىتىش
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => 1-9غىچە / ئالدىقى / كېيىنكى تەرجىمە مۇلازىمىتىگە ئالماشتۇرۇش 
Shift+Esc => ئاپتوماتىك پەرقلەندۈرگىلى قايتا تىل بىرىكمىسى بېكىتىش
Ctrl+Shift+1..9 => ‎تىل بىرىكمىسى تاللاش
Ctrl+I => تەرجىمە يۆنىلىشىنى ئالماشتۇرۇش
Ctrl+بوشلۇق => ‎‎ئاپتوماتىك تاماملاش / پىكىر تىزىملىكىنى كۆرسىتىش
F1 => ياردەمنى كۆرسىتىش
Alt+سول ئسترېلكا => ‎ئالدىنقى تەرجىمىگە يۆتكەش
Alt+ئوڭ ئسترېلكا => ‎كېيىنكى تەرجىمىگە يۆتكەش
Ctrl+ئۈستى ئىسترېلكا => ‎تەرجىمىنى تېكىست كىرگۈزۈش رامكىسىغا كۆچۈرۈش

ئىككى خىل مائۇستا تاللايدىغان تەرجىمە ھالىتى بار:
  1)سىنبەلگە كۆرسىتىش ‎(مەزكۇر پىروگراممىنىڭ سىنبەلگىسى مائۇس ئىسترېلكىسى يېنىدا كۆرۈنىدۇ، شۇ سىنبەلگىنى چەكسە، سەكرىمە كۆزنەكتە تاللىغان تېكىستنىڭ تەرجىمىسىنى كۆرسىتىپ بېرىدۇ - پىروگراممىدىن تېكىست تاللاش);
  2) تەرجىمىنى كۆرسىتىش ‎(پىروگراممىدىن تېكىست تاللاش - ‎تاللىغان تېكىستنىڭ تەرجىمىسىنى سەكرىمە كۆزنەكتە كۆرسىتىپ بېرىدۇ).

مەشغۇلاتلار:
  - سىستېما رول تاختىسىدىكى پىروگرامما سىنبەلگىسىنى چېكىپ، مائۇس تاللاش ھالىتىنى ئاچقىلى/ئەتكىلى بولىدۇ (ئۈستىدىكى مەزمۇنغا قاراڭ);
  - سىستېما رول تاختىسىدىكى پىروگرامما سىنبەلگىسىنى قوش چەكسە ئاساسىي كۆزنەك كۆرۈنىدۇ;
  - سەكرىمە كۆزنەكنىڭ ماۋزۇ ئىستونىنى قوش چەكسە ئاساسىي كۆزنەك نۆۋەتتىكى تەرجىمە بىلەن ئېچىلىدۇ;
  - سەكرىمە كۆزنەكتىكى مۇلازىمەت نامىنى چەكسە، نۆۋەتتىكى مۇلازىمەتچى ئۆزگىرىدۇ ۋە تېكىستنى قايتا تەرجىمە قىلىدۇ.
