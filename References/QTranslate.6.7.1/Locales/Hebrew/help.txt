מקשי קיצור גלובלים (ערכי ברירת מחדל ):
Double Ctrl => הצג בחלון ראשי (עם תרגום של טקסט נבחר, אם בכלל)
Ctrl+Q => תרגם טקסט שנבחר ולהראות את זה בחלון קופץ
Ctrl+Shift+Q => הצג חלון למילון
Ctrl+E => הקשב לטקסט נבחר

מקשי קיצור חלון ראשי:
Ctrl+Enter => תרגם טקסט
Ctrl+N => נקה תרגום נוכחי
Ctrl+D => הצג מילון
Ctrl+Alt+1..9 => הצג מילון עם שירות תרגום 1..9 שנבחר
Ctrl+H => הצג היסטורית תרגום
Ctrl+K => הצג מקלדת וירטואלית
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => עבור להבא / הקודם / שתרות תירגום 1..9
Shift+Esc => אפס את השפה לזיהוי אוטומטי
Ctrl+Shift+1..9 => בחירת זוג לשפה
Ctrl+I => החלף כיוון תרגום
Ctrl+Space => הצג הצעה / תפריט השלמה אוטומטית
F1 => הצג עזרה
Alt+Left Arrow => עבור אל התרגום הקודם
Alt+Right Arrow => עבור אל התרגום הבא
Ctrl+Up => העתק תרגום לתיבת הטקסט

ישנם שלושה מצבים של תרגום על ידי בחירת עכבר:
  1) הצג סמל  (בחר טקסט מתוך ישום -> סמל QTranslate יופיע סמוך לסמן. לחץ על הסמל -> חלון קופץ עם התרגום של טקסט נבחר יופיע);
  2) הצג תרגום (טקסט נבחר בישום -> חלון קופץ עם תרגום של הטקסט הנבחר יופיע במידי);
  3) הצג תרגום וקרא (זהה לקודם בתוספת דיבור טקסט נבחר בקול רם). 

פעולות:
  - לחצן שמאלי על סמל QTranslate באזור התראות הופך פעיל / לא פעיל מצב בחירת עכבר (ראה לעיל); 
  - לחץ לחיצה כפולה על QTranslate הסמל באזור התראות להצגת חלון הראשי;
  - לחיצה כפולה על כותרת חלון קופץ  פותח את החלון הראשי עם התרגום הנוכחי;
  - לחיצה על מקש השמאלי  של העכבר על השם של שירות בחלון הקופץ משנה את השירות הנוכחי מחדש את תרגם הטקסט.
